# TypeScript/React Frontend Utilities (Legacy)

This is the TypeScript/React frontend integration package from ATAuth v1.x.

**Status: Unmaintained.** Apps should use standard OIDC client libraries (e.g., `oidc-client-ts`, `next-auth`, or your framework's built-in OIDC support) instead of this package.

For current usage, see the [gateway/](../gateway/) directory.
